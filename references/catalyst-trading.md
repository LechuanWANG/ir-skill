# 催化、事件与交易研究

用于 3–6 个月催化、一个月内交易、宏观/行业事件、技术面、动量、因子筛选、入场节奏和价格异动。它不自动评价公司的长期价值。

## 选择时间路径

- **中期（约 3–6 个月）**：围绕可验证的盈利、订单、供需、政策、产品或估值催化，检查兑现窗口和价格是否已反映预期。
- **短期（一个月内）**：围绕已排期事件或明确市场状态，先判断行业与个股暴露，再用价格、成交、波动和流动性确认是否可执行。
- **1–3 个月重叠区间**：催化兑现主导时用中期；价格与事件执行主导时用短期，并说明假设。
- **纯技术/因子筛选**：可产生研究候选，但不能仅凭指标生成长期评级或个性化交易指令。

## 中期催化

1. 明确催化是什么、何时发生、市场何时能看到结果，以及用什么原始来源核验。
2. 区分事件发生、公开披露和价格反应；比较市场共识、当前定价和可能的预期差。
3. 检查催化如何改变收入、成本、利润率、现金流、估值或风险偏好。
4. 同时检查基本面兑现、估值变化、趋势、成交和流动性；过期催化不得继续支撑结论。
5. 写明催化未发生、数据不及预期、行业变量反转或价格条件恶化时的撤销条件。

## 中期财务核验门槛

对 3–6 个月的个股推荐、候选比较或“近期是否买入”问题，完成最小财务核验后才能给出 `优先行动`。这不是多年持有的完整基本面尽调，只检查会改变催化兑现、下行风险或候选排序的项目。

1. 取得最新年报及最新季报、半年报或已披露业绩预告的公司/交易所/巨潮原始披露；记录公告日、报告期、单位和币种。
2. 核验收入、归母利润或经营利润率、经营现金流及利润现金转化；按行业相关性检查应收/存货或合同负债，以及负债、资本开支或扩产。只讨论会改变当前催化的项目。
3. 把财务变化接回传导链：说明它通过销量、价格、成本、营运资本、产能或资本回报影响未来 3–6 个月预期的路径、时间窗口和反向情景。
4. TuShare 的 `income`、`balancesheet`、`cashflow` 和 `fina_indicator` 仅用于定位报告期、交叉检查或发现异常；报告中的财务事实和关键数字以原始披露为准。
5. 原始披露不可得、标准化接口为空或权限不足时，明确记录缺口。若不同财务结果可能改变排序、估值安全边际或行动标签，使用 `等待证据`，写明待核验文件、预期披露时间、正反结果及等待期间的替代选择；不得用价格和 PE/PB 代替。

## 短期传导链

短期荐股、候选比较或买入判断使用：

`已核实/已排期事件或市场状态 → 相对共识的预期差 → 利率/汇率/商品/流动性/风险偏好 → 行业与价值链环节 → 个股实际收入/成本/订单/估值/资金暴露 → 价格与成交确认 → 当前行动标签`

同时写出至少一条反向链。每个关键箭头说明证据、方向、时间窗口和未知项；事件日程可以是事实，事件结果和市场反应只能写成情景。

- 先比较受益、受损和影响不明确的行业，再核验个股；不要先选股票再倒推故事。
- 无法核验行业映射、关键箭头依赖传闻、公司实际暴露不足，或利好已充分定价时，不把事件链作为行动主因。
- 找不到可靠事件时不要虚构催化；把结果明确降级为价格/流动性筛选，并降低结论强度。

## 市场与技术确认

根据问题选择少量有经济含义的观察，不堆叠高度相关指标：

- 相对收益、趋势稳定性、回撤和波动；
- 成交参与、换手、流动性、资金和拥挤程度；
- 估值分位、行业与基准相对表现；
- 停牌、新股、复权、涨跌停、T+1、交易成本、滑点和隔夜跳空风险。

行业逻辑成立但价格结构或成交未确认时，可等待价格；价格走强但事件链、行业映射、业务暴露或中期财务核验未完成时，优先等待证据或回避。两条路径相互支持且风险收益优于现金或替代项时，才提高行动强度。

## 因子与系统化筛选

计算前明确股票池、`as_of`、数据可得时间、持有期、基准、复权、再平衡和交易约束。检查未来函数、幸存者偏差、参数敏感性、行业暴露、换手成本、样本外表现和极端行情依赖。未通过检查的因子只作研究线索。

本地指标可由 `scripts/tushare_mode_data.py indicators` 从已入库数据计算。MACD、RSI、布林带、均线、波动和量价指标都是观察项，不是固定阈值或自动信号。

## 异动归因与输出

价格异动按基本面、估值/情绪、资金/流动性和未知因素组织，说明证据和时间顺序。归因本身不自动回答是否买入；只有用户提出比较或行动问题时，才进入主 Skill 的决策纪律。

行动结论补充明确的触发条件、失效条件、最长观察/持有窗口和复核事件。用户未提供风险预算、持仓和交易成本时，不虚构精确仓位或止损比例。
