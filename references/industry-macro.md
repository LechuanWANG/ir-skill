# 行业与宏观

本文件用于生成行业和宏观证据。行业长期结构可以进入阶段 L；当前宏观、行业景气和市场反应属于 `research-screening.md` 的阶段 N。

## 时间尺度

每个结论先标明：

| 值 | 时间尺度 | 典型问题 |
|---|---|---|
| `structural_3_5y` | 3–5 年 | 行业容量、竞争结构、政策制度、技术替代 |
| `cyclical_6_24m` | 6–24 个月 | 库存、产能、价格、订单、信用和盈利周期 |
| `event_0_6m` | 0–6 个月 | 突发政策、地缘、供应中断、情绪与流动性 |

不要把短期顺风描述成长期护城河，也不要因一个季度逆风自动否决优质公司。

## 阶段 L：行业长期结构

- 长期需求、渗透率和可服务市场如何变化。
- 供给进入/退出壁垒、资本强度和历史资本回报。
- 谁拥有定价权，价值链哪个环节捕获经济利益。
- 行业集中度、竞争者反应、替代技术和监管结构如何变化。
- 公司各业务分部在价值链中的位置，长期增长是否需要不现实的份额假设。

阶段 L 的行业证据必须连接公司的收益来源、再投资空间和竞争优势，而不是按行业标签给分。

## 阶段 N：当前行业景气

- 供需、产能、库存、价格、订单和开工率处于什么阶段。
- 盈利改善来自真实需求、供给收缩、涨价、低基数还是短期补库。
- 竞争者是否会扩产、降价或融资，使利好无法持续。
- 政策对龙头、二线、上游和下游的影响是否不同。
- 公司是直接受益、间接受益，还是只有概念关联。
- 行业变化如何影响具体业务分部和利润敏感性。

## 阶段 N：宏观传导

只分析会改变具体长期假设或当前赔率的变量：

- 利率与流动性：融资成本、折现率、风险偏好和估值久期。
- 财政、产业、税收、补贴和监管：真实订单、成本、进入壁垒和竞争格局。
- 汇率、能源、原材料、航运和关税：收入、成本、毛利率和现金流。
- 通胀、PMI、信用和货币供应：公司盈利周期的领先、同步或滞后关系。
- 战争、制裁和供应链扰动：直接敞口、替代供应、库存和尾部风险。

宏观逆风不自动否决长期通过的公司，但必须影响情景概率、可接受价格、仓位节奏和安全边际。

## 证据输出契约

每项重要判断返回：

```text
thesis_id
time_horizon = structural_3_5y / cyclical_6_24m / event_0_6m
claim
transmission_channel
affected_metric = revenue / cost / margin / cashflow / WACC / risk_premium
direction = positive / negative / mixed / unknown
estimated_magnitude
confidence
source
source_date
as_of
unresolved_questions[]
```

无法映射 `thesis_id` 或财务传导的宏观评论只作为背景，不进入投资结论。

## 必需网络刷新

对宏观敏感结论，最终确定前运行当前网络检索：

- 政策、监管、补贴、关税、出口管制或制裁。
- 战争、地缘冲突、航运或供应链扰动。
- 央行决策、利率、流动性操作和信贷政策。
- 汇率、大宗商品、通胀、PMI、就业、GDP 和财政政策。

事实优先使用官方来源，以可靠财经媒体核对市场反应和解读。记录发布日期、事件日期、获取日期和冲突。无法联网时标记未验证、降低置信度，并通常使用 `wait_evidence`，不要用过期宏观状态给高确信结论。

## 价格异动归因

| 类型 | 含义 |
|---|---|
| 基本面 | 盈利、指引、产品、监管或竞争现实变化 |
| 情绪 | 叙事或风险偏好变化，长期假设未明显改变 |
| 资金/流动性 | 仓位、回购窗口、指数/ETF flow 或被动交易 |
| 未知 | 公开证据无法解释幅度 |
| 混合 | 多个原因叠加，给出有根据的粗略贡献 |

归因只解释发生了什么。若用户问“异动后能不能买”，先用 `news-intelligence.md` 核实事件，再进入双阶段状态机。未知或混合异动要保留不确定性和下一验证动作。
