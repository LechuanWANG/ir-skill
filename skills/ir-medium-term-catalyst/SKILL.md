# 中期催化研究

用于约 3–6 个月的盈利、订单、供需、政策、产品或估值催化研究，以及以催化兑现为主的 1–3 个月问题。先读取共享的研究纪律与项目边界：[`../shared/research-discipline.md`](../shared/research-discipline.md)。

## 共享支持材料

- 获取公司、交易所、监管、政府或行业机构的网页、PDF、原始披露或政策资料时，读取 [`../shared/external-evidence-sources.md`](../shared/external-evidence-sources.md)。
- 使用 TuShare、项目 SQLite 或结构化财务、宏观、行情和跨资产数据时，读取 [`../../references/tushare-data.md`](../../references/tushare-data.md)。
- 用户要求保存、复用、历史复盘或 Wiki，或任务确有多阶段、交接、长命令和上下文压缩风险时，读取 [`../../references/persistence.md`](../../references/persistence.md)。

若结论还依赖一个月内的事件执行、技术位置或入场节奏，同时调用 [`../ir-short-term-trading/SKILL.md`](../ir-short-term-trading/SKILL.md)。

## 研究方法

用于约 3–6 个月的盈利、订单、供需、政策、产品或估值催化研究，以及以催化兑现为主的 1–3 个月问题。它不自动评价公司的长期价值。若当前结论还依赖一个月内的事件执行、技术位置或入场节奏，同时读取 [`../ir-short-term-trading/SKILL.md`](../ir-short-term-trading/SKILL.md)。

### TuShare 数据调用包

先规划市场与催化数据，再补结构化财务基线：

```bash
python3 scripts/tushare_mode_data.py plan medium --symbol 000001.SZ --end-date 20260719
python3 scripts/tushare_research_data.py plan financial --symbol 000001.SZ --period 20260331 --as-of 20260719
python3 scripts/tushare_sector_data.py performance --provider ths --sector-type I --as-of 20260719 --sort-by return_20d
python3 scripts/tushare_sector_data.py memberships --provider ths --stock-code 000001.SZ --as-of 20260719
```

基础包覆盖价格、估值、资金、市场杠杆、业绩预告/快报和披露日历。按催化扩展券商预期、机构调研、行业资金、板块成分、龙虎榜和因子；按供给风险调用解禁、回购、大宗交易等公司行为数据。结构化三表与财务指标通过 `financial` family 获取。

先查本地缓存，再对缺口执行 `plan` 和定向 `fetch`。关键财务数字、订单和公司行为执行状态使用最新原始披露确认。完整路由见 [`../../references/tushare-data.md`](../../references/tushare-data.md)。

当行业轮动、政策传导、供需变化或板块资金会改变催化判断时，必须读取 `sector-type I` 的行业 `performance` 和个股 `memberships`；本地无数据或覆盖不足时先 `plan`，再用相同 `as_of` 定向 `fetch`。同花顺用于市场板块表现，申万 2021 用于基本面同行参照；二者冲突时保留冲突并核验公司真实业务暴露，不按标签多数投票。

### 中期催化

参考以下方面进行中期研究：

1. 明确催化是什么、何时发生、市场何时能看到结果，以及用什么原始来源核验。
2. 区分事件发生、公开披露和价格反应；比较市场共识、当前定价和可能的预期差。
3. 检查催化如何改变收入、成本、利润率、现金流、估值或风险偏好。
4. 同时检查基本面兑现、估值变化、趋势、成交和流动性；过期催化不得继续支撑结论。
5. 写明催化未发生、数据不及预期、行业变量反转或价格条件恶化时的撤销条件。

### 宏观与行业传导推断

首先考虑宏观与行业因素，当利率、汇率、商品、流动性、政策、需求、库存或行业供给等宏观变量可能影响未来 3–6 个月的催化兑现时，使用传导链形成可证伪的中期假设，例如：

`已核实的宏观/政策/行业状态 → 利率/汇率/商品/流动性/供需变化 → 行业与价值链环节 → 个股销量/价格/成本/订单/营运资本/资本开支暴露 → 收入、利润率、现金流、估值或风险偏好 → 催化兑现与当前行动标签`

- 宏观状态可以是事实，后续传导和市场反应只能写成假设或情景。
- 先比较受益、受损和影响不明确的行业或价值链环节，再核验个股实际暴露、经营杠杆和当前定价；不要先选股票再倒推宏观叙事。
- 宏观或行业映射无法核验、关键箭头依赖传闻、个股暴露不足或预期已充分反映时，不把传导链作为行动主因；缩小主张范围、降低行动强度或使用 `等待证据`。
- 传导链用于解释和验证中期催化，不替代原始披露、公司财务事实或下节的中期财务核验门槛。

### 财务核验门槛

对 3–6 个月的个股推荐、候选比较或“近期是否买入”问题，完成与决策风险相称的必要财务核验后才能给出 `优先行动`。这不是多年持有的完整基本面尽调，但必须覆盖可能改变催化兑现、下行风险或候选排序的项目；不得以研究时间有限为由省略其中任何一项。

1. 取得最新年报及最新季报、半年报或已披露业绩预告的公司/交易所/巨潮原始披露；记录公告日、报告期、单位和币种。
2. 核验收入、归母利润或经营利润率、经营现金流及利润现金转化；按行业相关性检查应收/存货或合同负债，以及负债、资本开支或扩产等重要财务内容。输出可不罗列无关项目，但不得遗漏可能改变当前催化、下行风险或候选排序的观察。
3. 把财务变化接回传导链：说明它通过销量、价格、成本、营运资本、产能或资本回报等影响未来 3–6 个月预期的路径、时间窗口和反向情景。
4. TuShare 的 `income`、`balancesheet`、`cashflow` 和 `fina_indicator` 仅用于定位报告期、交叉检查或发现异常；报告中的财务事实和关键数字以原始披露为准，并进行财务指标的适当扩充。
5. 原始披露不可得、标准化接口为空或权限不足时，明确记录缺口。若不同财务结果可能改变排序、估值安全边际或行动标签，使用 `等待证据`，写明待核验文件、预期披露时间、正反结果及等待期间的替代选择；不得用价格和 PE/PB 代替。

### 预期、变化与资金状态

- 对业绩和行业数据，不只比较同比或环比；同时比较实际值、市场共识和价格隐含预期，解释变化是否足以触发盈利或估值上修、下修。
- 同时判断水平、方向和加速度。高增长但持续放缓、低增长但边际改善、绝对利好但不及预期、绝对利空但好于预期，含义不同。
- 判断资金的水平、方向和加速度，并相对个股自身历史、行业和全市场定义高低位；区分增量流入、存量轮动、被动资金、回补和拥挤交易，不因高低位本身预设反转。
- 从全球利率、汇率、商品、流动性和风险偏好映射当前风格偏好与边际买方，再核验行业和个股暴露；宏观叙事不能替代实际传导链。
- 公开披露用于确认已发生的财务事实。价格、成交、资金和其他高频数据可作为前瞻线索，但只形成带时间戳、可证伪的假设；不得覆盖原始披露或绕过中期财务核验门槛。

### A股的个股供给与被动资金

当事件位于当前持有期，或合理预期可能改变候选排序、价格条件或行动标签时检查；不为堆叠信号而机械扩展，但也不得因追求简短而忽略已知的重大供给或被动资金事件。

- 核验已公告或即将发生的解禁、减持、再融资/配股、回购、质押处置和控制权变化。说明数量相对流通股和近期成交的大小、股东类型、执行窗口、价格条件及实际完成进度；计划不等于执行，回购不等于价格底。
- 只有个股存在已公告的指数调整，或在规模足以影响其流动性的 ETF 中具有可查权重时，才估算被动资金影响；同时核验公告日、实施日、权重、基金份额或规模、前期价格反应和成交确认。指数纳入/剔除、解禁或机构持仓本身不能单独支持行动结论。
- 不默认推断主动公募实时换仓：基金和股东定期披露只证明报告期末状态；价格、成交和板块表现只能形成带时间戳的备选假设。

### 输出

行动结论补充明确的催化触发条件、失效条件、最长观察/持有窗口和复核事件。用户未提供风险预算、持仓和交易成本时，不虚构精确仓位或止损比例。
