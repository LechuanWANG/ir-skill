# 长期与基本面研究

用于多年持有、单股基本面、商业质量、财务质量、治理、资本配置和长期估值。先读取共享的研究纪律与项目边界：[`../shared/research-discipline.md`](../shared/research-discipline.md)。

## 共享支持材料

- 获取公司、交易所、监管、政府或行业机构的网页、PDF、原始披露或政策资料时，读取 [`../shared/external-evidence-sources.md`](../shared/external-evidence-sources.md)。
- 使用 TuShare、项目 SQLite 或结构化财务、宏观、行情和跨资产数据时，读取 [`../../references/tushare-data.md`](../../references/tushare-data.md)。
- 用户要求保存、复用、历史复盘或 Wiki，或任务确有多阶段、交接、长命令和上下文压缩风险时，读取 [`../../references/persistence.md`](../../references/persistence.md)。

对非股票资产，将本 Skill 中的公司基本面分析替换为适用的标的、合约、基准和宏观传导核验。

## 研究方法

用于多年持有、单股基本面、商业质量、财务质量、治理、资本配置和长期估值。3–6 个月催化使用 [`../ir-medium-term-catalyst/SKILL.md`](../ir-medium-term-catalyst/SKILL.md)；一个月内的事件、技术或交易筛选使用 [`../ir-short-term-trading/SKILL.md`](../ir-short-term-trading/SKILL.md)。

### TuShare 数据调用包

先规划市场与公司行为数据，再补结构化财务历史：

```bash
python3 scripts/tushare_mode_data.py plan long --symbol 000001.SZ --end-date 20260719
python3 scripts/tushare_research_data.py plan financial --symbol 000001.SZ --period 20251231 --as-of 20260719
```

基础包覆盖长期价格、复权、基准、估值、交易日历、公司信息和分红。按问题扩展行业分类、指数权重、股东结构、解禁、质押、回购、增减持、大宗交易和名称变化。结构化三表、财务指标、审计意见和主营构成通过 `financial` family 获取；宏观与跨资产数据按商业驱动调用相应 family。

长期同行比较继续以申万 2021 等稳定行业分类为基本面参照。只有当前估值、风险偏好或买入时点确实依赖市场板块状态时，补充调用 `scripts/tushare_sector_data.py performance --provider ths --as-of <日期>`；同花顺板块强弱只能解释定价环境，不能直接改写长期行业边界、竞争优势或现金流判断。

先复用本地历史，再补缺口。最终财务、治理和资本配置事实使用定期报告与公告确认。记录 `as_of`、数据版本、权限状态和来源冲突。完整路由见 [`../../references/tushare-data.md`](../../references/tushare-data.md)。

### 研究重点

围绕当前决策选择最能改变排序、估值和主要风险的维度，不执行与标的无关的固定清单式尽调。研究可以聚焦，但覆盖范围必须与不确定性、下行风险和结论强度相称；不得为了缩短研究而排除潜在的决定性维度。

#### 生意与需求

- 公司卖什么、谁付钱、为什么持续付钱；哪些分部真正贡献收入、利润和现金流。
- 需求来自长期经济价值，还是补贴、周期补库、单一客户或一次性项目。
- 收入模式、议价权、客户留存、客户集中度和单位经济性是否可持续。

#### 增长、竞争与再投资

- 行业容量、渗透率、份额、产品扩张和国际化空间。
- 同时判断规模和回报的当前水平、增长方向、增速变化与持续期；区分高增长但边际恶化、低增长但拐点改善、一次性跳升和可持续复利。
- 增长需要的资本、营运资金和杠杆；边际资本回报是否下降。
- 护城河来自成本、品牌、渠道、技术、牌照、网络效应还是客户黏性；竞争者复制需要多久和多少资本。
- 扩产、降价、技术替代、政策或周期如何破坏当前逻辑。

#### 财务质量

从原始定期报告、附注和公告检查：

- 收入、利润、毛利率、经营利润率、ROIC/ROE 及其驱动。
- 经营现金流、自由现金流、应收、存货、合同负债和利润现金转化。
- 三张表是否相互印证；非经常项目、资本化、收入确认和会计估计是否扭曲利润。
- 增长来自销量、价格、结构、成本、补贴还是会计处理；区分持续改善和一次性贡献。
- 财务披露用于确认已发生的事实；未来路径优先用订单、价格与销量、客户预算、合同负债、产能利用率等行业相关领先指标验证，来源较弱时只作假设并降低置信度。

#### 资本配置、治理与尾部风险

- 研发、扩产、并购、商誉、分红、回购、融资、稀释和股东回报记录。
- 管理层承诺兑现、激励、关联交易、担保、资金占用和控股股东利益一致性。
- 减持、质押、诉讼、监管问询、审计意见、会计更正和审计机构变更。
- 欺诈、偿债、流动性或重大合规风险是否可能造成永久性损失。

#### 估值与隐含预期

- 将当前价格转换为对收入增长、利润率、资本回报和再投资期的隐含要求。
- 比较这些要求与公司历史、行业容量、竞争反应、市场共识和可观察的领先指标，明确预期差及其最早验证时间。
- 给出合理区间或情景敏感性，不只陈述 PE/PB 高低。
- 区分基本面价值与资金驱动的价格波动；资金和情绪主要影响行动时点、安全边际和估值波动，除非其通过融资成本、需求或生存能力改变未来现金流，否则不直接改写长期价值。

### 可决策结论

完成研究时至少说明以下内容；如某项无法核验，明确其对结论的影响，而不是以概括性风险提示代替：

- 两至四个决定未来价值的核心变量；
- 当前判断依赖的关键假设及最强反证；
- 最早显示改善或恶化的领先指标；
- 哪项事实会改变排序、估值区间或行动标签；
- 当前价格相对价值、现金和其他候选是否提供足够安全边际。

历史不足或关键数据不可得时，缩小主张范围并降低置信度；不要用统一评分或一般性风险清单代替因果解释。
